JOB TITLE: Python Developer - Machine Learning & Data Science

COMPANY: IntelliSoft Technologies Pvt Ltd
LOCATION: Pune, Maharashtra (Hybrid)
EXPERIENCE: 1-3 Years

JOB DESCRIPTION:
We are looking for a passionate Python Developer with hands-on experience in
machine learning, data analysis, and API development. The ideal candidate should
have strong problem-solving skills and the ability to work in an Agile environment.

KEY RESPONSIBILITIES:
- Develop and deploy machine learning models using Python and Scikit-learn
- Build and maintain RESTful APIs using Flask or Django
- Perform data analysis and preprocessing using Pandas and NumPy
- Collaborate with data scientists to implement NLP and deep learning solutions
- Work with databases like MySQL and MongoDB
- Deploy models to cloud platforms (AWS or GCP)
- Write clean, well-documented, and testable Python code
- Participate in code reviews and Agile sprint planning

REQUIRED SKILLS:
Technical Skills:
- Strong proficiency in Python programming
- Experience with machine learning libraries: Scikit-learn, TensorFlow or PyTorch
- Data manipulation using Pandas and NumPy
- Web frameworks: Flask or Django
- SQL and NoSQL databases
- Version control using Git
- Basic knowledge of Docker and cloud deployment

Preferred Skills:
- Experience with NLP (Natural Language Processing)
- Knowledge of deep learning frameworks
- Familiarity with AWS or Google Cloud Platform
- Understanding of data visualization tools

EDUCATIONAL QUALIFICATIONS:
- B.Tech / B.E. / M.Tech / M.Sc in Computer Science, IT, or related field
- Minimum 60% in graduation

SOFT SKILLS REQUIRED:
- Strong analytical and problem-solving ability
- Good communication skills
- Ability to work in a team (Agile/Scrum)
- Self-motivated and eager to learn

COMPENSATION: 4-8 LPA based on experience
